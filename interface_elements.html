Elements Documentation

Global Attributes

:

  • id - Defines a unique identifier for the element, and can be used for specific manipulation of the element using JavaScript.
  • style - Define inline vbkit style rules directly on the element
  • vbkit.scrollable - Enables or disables the scrollbar for element:
    "enable-all" - Enables horizontal and vertical scrollbars.
    "enable-horizontal-only" - Enables horizontal scrollbar only.
    "enable-vertical-only" - Enables vertical scrollbar only.
    "disabled" - Does not include a scrollbar for that element (default value).
  • vbkit.resizable - Enables or disables the scrollbar for element:
    "enable-all" - Enables horizontal and vertical scrollbars.
    "enable-horizontal-only" - Enables horizontal scrollbar only.
    "enable-vertical-only" - Enables vertical scrollbar only.
    "disabled" - Does not include a scrollbar for that element (default value).
  • ignore - Defines whether or not the element should be ignored:
    "non-ignorable" or omitted will always display the element,
    "force-ignored" will hide the element, reserving the original space in the layout,
    "ignored" hides the element without reserving original space in the layout.

label

Creates a viewable frame, which is used to group other elements or associated with a form element id (if thefor attribute is provided), similar to creating a <div> in HTML.

XML Attributes:

  • for - Associates the label with a form element id.
  • width - Defines the width of label element, which can be; static: using pixel values, or relative: using relative percentage values.
  • height - Defines the height of label element, which can be; static: using pixel values, or relative: using relative percentage values.

XML:

<label id="my_label" width="fill" height="fill">
</label>

stackLabel

Derived from <label>, Creates a viewable frame which is used to group other elements, and arranges child elements in a single direction defined by the orientation value.

XML Attributes:

  • orientation - Set stacking orientation of child elements:
    "vertical" - Stacks child elements vertically; (default value)
    "horizontal" - Stacks child elements horizontally;
  • width - Defines the width of stackLabel element, which can be; static: using pixel values, or relative: using relative percentage values.
  • height - Defines the height of stackLabel element, which can be; static: using pixel values, or relative: using relative percentage values.

XML:

<stackLabel id="my_stack_label" orientation="vertical" width="fill" height="fill">
</stackLabel>

gridLabel

Derived from <label>, Creates a viewable frame arranged in grid format, allowing the organization of elements in rows and columns.

XML Attributes:

  • width - Defines the width of gridLabel element, which can be; static: using pixel values, or relative: using relative percentage values.
  • height - Defines the height of gridLabel element, which can be; static: using pixel values, or relative: using relative percentage values.

XML:

<gridLabel id="my_grid_label" width="150px" height="100px">
</gridLabel>

gr

Adds new row to a grid created using <gridLabel> element.

XML Attributes:

  • width [NOT RECOMMENDED] - Defines the width of gr element, which can be; static: using pixel values, or relative: using relative percentage values.
  • height - Defines the height of gr element, which can be; static: using pixel values, or relative: using relative percentage values.

XML:

<gridLabel id="my_grid_label" width="100px" height="100px">
  <gr>

  </gr>
<gridLabel>

gridContent

Adds new cell within a row in a grid created using <gridLabel>, allowing the arrangement of elements.

XML Attributes:

  • width - Defines the width of gridContent element, which can be; static: using pixel values, or relative: using relative percentage values.
  • height [NOT RECOMMENDED] - Defines the height of gridContent element, which can be; static: using pixel values, or relative: using relative percentage values.

XML:

<gridLabel id="my_grid_label" width="100px" height="100px">
  <gr>
    <gridColumn>

    <gridColumn>
  </gr>
<gridLabel>

textLabel

Derived from <label>, Creates a text view, similar to creating a <p> or <span> in HTML.

XML Attributes:

XML:

<textLabel id="my_text">Hello World!</textLabel>

imageLabel

Derived from <label>, Creates an image view, with image content set using require value.

XML Attributes:

  • import [DEPRECATED] - Name of imported resource using <import> to set image content.
  • require - Name of imported resource using <import> to set image content.
  • width - Defines the width of imageLabel element, which can be; static: using pixel values, or relative: using relative percentage values.
  • height - Defines the height of imageLabel element, which can be; static: using pixel values, or relative: using relative percentage values. If omitted, width value is used to adjust size proportionally.

XML:

<imageLabel id="my_image" width="350px" height="280px" require="flowers1">
</imageLabel>

objectLabel

Creates an file data view, with source content set using require attribute value.

XML Attributes:

  • import [DEPRECATED] - Name of imported resource using <import> to set image content.
  • require - Name of imported resource using <import> to set image content.
  • width - Defines the width of objectLabel element, which can be; static: using pixel values, or relative: using relative percentage values.
  • height - Defines the height of objectLabel element, which can be; static: using pixel values, or relative: using relative percentage values.

XML:

<objectLabel id="my_object" width="150px" height="95px" require="object_page">
</objectLabel>

textbox

An input to set text or number.

XML Attributes:

  • type - Determines the accepted type of input value:
    "text" - Accepts plain text (default value),
    "number" - Accepts number.
  • value - The default value for input.
  • placeholder [NEW] - The text displayed when input is empty.
  • width - Defines the width of textbox element, which can be; static: using pixel values, or relative: using relative percentage values.
  • height - Defines the height of textbox element, which can be; static: using pixel values, or relative: using relative percentage values.

XML:

<textbox id="my_textbox" type="number" placeholder="Type...">
</textbox>

button

Creates an interactive button to trigger an event callback.

XML Attributes:

  • action - [REFORMING] Sets a default action for the button:
    "default" sets no action,
    "submit" sends data from all fields in a form,
    "reset" resets the values of all fields to the default in a form.
    This may be changed soon.
  • width - Defines the width of button element, which can be; static: using pixel values, or relative: using relative percentage values.
  • height - Defines the height of button element, which can be; static: using pixel values, or relative: using relative percentage values.

Note: You can add interactivity to button using JavaScript.

XML:

<button id="my_button" width="50px" height="28px" action="default">
</button>
